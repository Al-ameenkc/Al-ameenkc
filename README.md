### 3. Contribution Snake Animation
To make the snake work, add a GitHub Action in your profile repo.

Create `.github/workflows/snake.yml`:

```yaml
name: Generate Snake

on:
  schedule:
    - cron: "0 0 * * *"
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: Al-ameenkc
          outputs: |
            dist/github-contribution-grid-snake.svg
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark

      - uses: crazy-max/ghaction-github-pages@v4
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

Then keep this image in your README:

```md
![snake](https://raw.githubusercontent.com/Al-ameenkc/Al-ameenkc/output/github-contribution-grid-snake-dark.svg)
```
